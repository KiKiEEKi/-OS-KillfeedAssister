Поддерживаемые игры:

CS: Source (OrangeBox), CS: Source (v34)

Описание:

Показывает ассисты в киллфиде

<img width="291" height="274" alt="изображение" src="https://github.com/user-attachments/assets/e4868bab-68e5-44f6-b06d-6c42a1eb757b" />

Требования:

Sourcemod (компилированный на 1.13, можно скомпилить на своей версии 1.9 или выше)

Возможность добавлять ботов на сервер (Арена этим просто грешит)

Переменные В исходнике:

float g_iEventTime = 0.5; //Через сколько сек. создать евент с ассистом

Установка:

Закинуть файл [OS]KillfeedAssister.smx в addons\sourcemod\plugins\
